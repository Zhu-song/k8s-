# 一、Helm
- K8s的**包管理工具**
- 把一堆YAML打包成**Chart**
- 一键安装、升级、卸载应用
- 解决：YAML太多、管理麻烦
# 二、Prometheus + Grafana
- **Prometheus**：监控采集、存储、告警
- **Grafana**：可视化、做大盘
- 监控：节点、容器、Pod、服务状态
- 是K8s**标准监控方案**
# 三、EFK/ELK日志
- E：**Elasticsearch** 存日志
- F：**Fluentd / Filebeat** 收集日志
- K：**Kibana** 查日志、展示日志
- 用于：统一收集、查询、排查日志
# 四、CI/CD+K8s
- 代码提交 → 自动构建镜像 → 自动部署到 K8s
- 常用工具：GitLab CI、Jenkins、ArgoCD
- 实现：**自动化发布、回滚、灰度发布**
# 五、容器镜像仓库
- 存放Docker镜像
- 常用：Harbor、Docker Hub、阿里云镜像仓库
- 作用：拉取、推送、管理镜像
# 六、Service Mesh 服务网格（了解）
- 管理微服务之间的流量、安全、可观测
- 代表：**Istio**
- 功能：灰度、限流、熔断、追踪
